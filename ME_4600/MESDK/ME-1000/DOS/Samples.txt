In this directory are the following simple examples:

VC152C_1 General Example for use with a DOS 
	operating system. The program uses BIOS calls
	to determine the hardware address and interrupt 
	line. It was tested with MS Visual C 1.52C
BC01	General Example for use with a DOS 
	operating system. The program uses BIOS calls
	to determine the hardware address and interrupt 
	line. It was tested with Borland C 3.1 only.

The example programs are shiped as is and Meilhaus 
Electronic offers no guarantees.
