In this directory is the following simple example:

sample01	General Example for ME-96 
sample02	General Example for ME-95
sample03	General Example for ME-94
sample04	General Example 
sample05	Example for me9xGetIrqCnt
sample06	Example Irq counter with dynamic DLL access


This are not full working samples, they should show
how you can access the board in a C environment.

The projects are written for Visual C 6.0 or 
Borland C Builder 4 only. (But should run also 
under higher versions)

If you have improvement proposals for the sample,
please send it to software@meilhaus.de

The example programs are shiped as is and Meilhaus 
Electronic offers no guarantees.
