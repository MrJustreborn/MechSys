Purpose:
	This program reads the PCI settings 
	of all PCI boards

Using requirements:
	This utility runs only under Windows NT
	(Windows NT 4, Windows 2000)

Serial number:
	You will find the serial number in the 
	field "Reserved x2C" in the lower centre 
	of the application view.
